Install instructions

Snowshoe is available for Nokia N9 through its own repository, which also hosts Qt5 and QtWebKit packages. To install, follow these steps:

Step 1 - Enable developer mode

You can find how to do this on MeeGo 1.2 Harmattan Developer Documentation.

Go to Settings -> Security -> Developer Mode and install Utilities, this will install wget (required by the repository installer).

Step 2 - Install repository package

Snowshoe is built using a recent QtWebKit revision and the latest Qt5 hashes announced on QtWebKit mailing list, to install all dependencies first download and isntall the repositories package, this package will add Snowshoe repositories to your N9, then you can install Snowshoe.

http://snowshoe.openbossa.org/apt/snowshoe-repositories.deb

Step 3 - Install Snowshoe

Use some package manager like MeeCatalog to install snowshoe or open a terminal as root and install it.

# apt-get install snowshoe-mobile

Compiling from sources

If you already have a scratchbox with a Harmattan setup on your machine is very easy to compile and run snowshoe on your n9, first select the ARMEL target on scratchbox, install the repository package (step 2), all dependencies (webkit-snapshot and qt-components2 packages) then go to snowshoe directory outsite scratchbox and type:

$ ./tools/build-deploy.sh -i <device ip> -r

The build-deploy.sh script provide more options, like generate packages, just compile, etc.
